# Northstar Planning Demo

This repository contains a lightweight React + TypeScript + Vite demo that recreates the feel of the Cyris workflow using Steppe Change-inspired branding and fully fictional automotive data.

## Current Scope

- `Performance` page for aggregated historical media results
- `Build a Plan` page for prompt-driven planning inputs
- `Plans` page for the generated recommendation
- Local-only dummy data with no backend dependencies

## Current Demo Story

The demo shows an automotive planning workflow for `Northstar Electric Motors` and the `Northstar Eon` launch:

1. Review historical automotive performance in `Performance`
2. Enter campaign prompts in `Build a Plan`
3. Wait through a simulated real-time generation state
4. Review the generated outcome in `Plans`

## Recent Change

The `Plans` outcome page no longer includes the 10 small summary widgets that previously sat under the campaign-level graphs. The page now favors larger, easier-to-read sections.
